## npm 命令

```shell
# 查询Node.js版本，辅验证是否安装成功
node -v
# 初始化
npm init --yes
# 安装express框架
npm i express
# 启动node服务端
node xxx.js
# 安装nodemon自动重启包
npm install -g nodemon
# 启动nodemon服务端
nodemon xxx.js
```
